---
title: ולנטינו
---

| במקור                     | וצריך להיות            |
|---------------------------|------------------------|
| מי היה מאמין,             |                        |
| שהיית פשוט,               |                        |
| שהיית תמים                | שהיית תנין             |

| במקור                     | וצריך להיות            |
|---------------------------|------------------------|
| שהיית בן אוהב של אמא      |                        |
| וחלמת באמת                |                        |
| על פרדס וקלמנטינות        | על פרדס בהירושימה      |

| במקור                     | וצריך להיות            |
|---------------------------|------------------------|
| הם ראו בך נסיך            |                        |
| הם ראו אותך בשיא          | הם ראו אותך על סִיח     |

| במקור                     | וצריך להיות            |
|---------------------------|------------------------|
| הם ראו והאמינוּ,           |                        |
| ולנטינוֹ                   | ולנטינוּ                |
